⚙️ Configuración inicial

Para guardar tus registros en la nube, necesitas conectar una base de datos Supabase gratuita.

Pasos rápidos:
1. Ve a supabase.com → crear cuenta gratis
2. Crear nuevo proyecto
3. Ir a SQL Editor y ejecutar el script de abajo
4. Copiar tu Project URL y anon key desde Settings → API
